# microGPT implementation

- [microgpt.ml](microgpt.ml): main file
- [autograd.ml](autograd.ml): autograding library
- [extlib.ml](extlib.ml): helper functions which ought to be in the standard library
- [backcomp.ml](backcomp.ml): functions from recent versions of the standard library which are copied in order to run with the OCaml version present on github's CIs
- [microgpt.py](microgpt.py): original reference python implementation
